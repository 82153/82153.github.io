[문제](https://www.acmicpc.net/problem/1406)
```python
def star(x, y):
    while x > 0 and y > 0:
        if x % 3 == 1 and y % 3 == 1:
            return False  # 정가운데일 경우만 별을 찍지 않음
        x //= 3
        y //= 3
    return True

n = int(input())
for i in range(n):
    for j in range(n):
        print('*' if star(i, j) else ' ', end='')
    print()
```
