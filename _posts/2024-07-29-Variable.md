### 변수
- 가장 기초적인 프로그래밍 문법 개념
- 데이터(값)을 저장하기 위한 메모리 공간의 프로그래밍상 이름


```python
# ex)
a = 5
# a라는 변수에 5라는 값을 넣으라는 의미
```

###### 변수는 <span style = 'color:#0000FF'>메모리 주소</span>를 가지고 있고 변수에 들어가는 <span style = 'color:#0000FF'>값</span>은 <span style = 'color:#0000FF'>메모리 주소</span>에 할당된다.

###### 즉, 변수가 선언되는 순간 <span style = 'color:#0000FF'>메모리 특정영역에 물리적인 공간</span>이 할당된다.

###### 좌변은 <span style = 'color:#0000FF'>저장되는 <u>공간</u></span> 우변은 <span style = 'color:#0000FF'>저장되는 <u>값</u></span>이다.

### Dynamic Typing
- 코드 실행시점에 데이터의 Type을 결정하는 방법
- JAVA나 C같이 int 같은 것으로 먼저 선언하지 않음
- 프로그래밍은 쉽지만 속도가 느려짐


```python
# 정수형
a = 1
print(type(a))

# 실수형
a= 1.5
print(type(a))

# 문자형
a = 'dfjkdjfk'
print(type(a))

# 불린형
a = True
print(type(a))
```

    <class 'int'>
    <class 'float'>
    <class 'str'>
    <class 'bool'>
    

### 데이터 형변환
- 단순히 float(a)를 한다고 a의 형이 변하지 않음
- a = float(a)를 해줘야 변함
- 실수형에서 정수형 변환시 반올림이 아니라, <span style = 'color:#0000FF'>소수점 이하는 없앰</span>


```python
a = 10
print(a)
print(type(a))
print()

print(float(a))
print(a)
print(type(a))
print()

a = float(a)
print(a)
print(type(a))
```

    10
    <class 'int'>
    
    10.0
    10
    <class 'int'>
    
    10.0
    <class 'float'>
    

### List or Array
- 시퀀스 자료형, 여러 데이터들의 집합
- int, float 같은 다양한 데이터 타입 포함
- python에서 list는 list안에 list를 넣는 것도 가능하다

###### 인덱싱
- list에 있는 <span style = 'color:#0000FF'>주소(offset)</span>를 가짐
- 주소를 사용해 할당된 값 호출


```python
names = ['kim', 'lee', 'park']
print(names[0])
print(names[2])
print(len(names))
```

    kim
    park
    3
    

###### 슬라이싱
- list의 값들을 잘라서 쓰는 것
- list의 주소 값을 기반으로 부분 값을 반환


```python
nums = [1, 45, 23, 5, 0, 3, 2, 11, 38]

# 0 ~ 5번, -4부터 끝까지
print(nums[0:6], "AND", nums[-4:])

# nums의 처음부터 끝까지
print(nums[:])

# 범위를 넘어가면 자동으로 최대 범위를 지정
print(nums[-100:100])

# 2칸 단위로 슬라이싱
print(nums[::2])

# 역으로 슬라이싱
print(nums[::-1])
```

    [1, 45, 23, 5, 0, 3] AND [3, 2, 11, 38]
    [1, 45, 23, 5, 0, 3, 2, 11, 38]
    [1, 45, 23, 5, 0, 3, 2, 11, 38]
    [1, 23, 0, 2, 38]
    [38, 11, 2, 3, 0, 5, 23, 45, 1]
    

### 리스트 연산
- 리스트끼리의 덧셈은 두 리스트를 합친다.
- 리스트에 숫자를 곱하면 그 리스트를 숫자만큼 반복


```python
color = ['red', 'white']
color2 = ['blue', 'black']

print(color+color2)
print(color*3)
```

    ['red', 'white', 'blue', 'black']
    ['red', 'white', 'red', 'white', 'red', 'white']
    

### 리스트 연산자
1. in: 해당 원소가 list에 존재 여부 반환
2. append: list의 맨 뒤에 원소 추가
3. extend: list에 새로운 list추가
4. insert: 특정 주소에 새로운 값 추가
5. remove: list에 특정 값 삭제
6. del: 특정 주소 list 객체 삭제
- 위의 연산자들은 모두 원본 list를 변경시킨다.


```python
color = ['yellow', 'blue', 'green']

print('blue' in color, end='\n\n')

color.append('white')
print(color, end = '\n\n')

color.insert(0, 'orange')
print(color, end = '\n\n')

color.extend(['black', 'purple'])
print(color, end = '\n\n')

color.remove('white')
print(color, end = '\n\n')

del color[0]
print(color, end = '\n\n')
```

    True
    
    ['yellow', 'blue', 'green', 'white']
    
    ['orange', 'yellow', 'blue', 'green', 'white']
    
    ['orange', 'yellow', 'blue', 'green', 'white', 'black', 'purple']
    
    ['orange', 'yellow', 'blue', 'green', 'black', 'purple']
    
    ['yellow', 'blue', 'green', 'black', 'purple']

### 팩킹, 언팩킹
- 팩킹: 한 변수에 여러 개의 데이터를 넣는 것
- 언팩킹: 한 변수의 데이터를 각각의 변수로 변환


```python
t = [1,2,3]

a,b,c = t
print(a,b,c)
```

    1 2 3
    


```python
kor = [39, 59, 100]
math = [97, 85, 88]
eng = [94,83, 54]

midterm = [kor, math, eng]
print(midterm)
```

    [[39, 59, 100], [97, 85, 88], [94, 83, 54]]
    


```python
mid_copy = midterm[:]
midterm[0][0] = 100

print(mid_copy)
print(midterm)
# 일차원 배열 처럼 값만 복사하는 것이 불가능하다.
```

    [[100, 59, 100], [97, 85, 88], [94, 83, 54]]
    [[100, 59, 100], [97, 85, 88], [94, 83, 54]]
    


```python
# copy라는 모듈을 사용해서 값만 복사

import copy
mid_copy = copy.deepcopy(midterm)
mid_copy[0][0] = 50

print('복사본:',mid_copy)
print('원본:',midterm)
```

    복사본: [[50, 59, 100], [97, 85, 88], [94, 83, 54]]
    원본: [[100, 59, 100], [97, 85, 88], [94, 83, 54]]
